# Checklist: Flow Quality

**ID:** ai-studio/flow-quality
**Version:** 1.0.0

---

## Pre-Design (Antes de comecar)

- [ ] Problema claro e especifico identificado
- [ ] Target user definido
- [ ] Alinhamento com Zona de Genialidade confirmado
- [ ] Fluxo nao duplica outro existente no portfolio

## Design (QG-DESIGN)

- [ ] Arquitetura com 3-7 steps atomicos
- [ ] Input/output documentado por step
- [ ] Prompt pattern selecionado por step
- [ ] Conexoes entre steps mapeadas
- [ ] Categoria e complexidade definidos
- [ ] Inacio aprovou o design

## Build (QG-BUILD)

- [ ] System prompt claro por step
- [ ] User prompt template com variaveis
- [ ] Output format especificado
- [ ] Model config definido (recomendado + fallback)
- [ ] Guardrails contra output indesejado
- [ ] Self-test executado com sucesso

## Test (QG-TEST)

- [ ] Happy path testado e aprovado
- [ ] Input minimo testado
- [ ] Edge cases criticos cobertos
- [ ] Reproducibilidade verificada (3 runs)
- [ ] Metricas de qualidade >= 85% media
- [ ] Nenhum P1 com status FAIL

## Package (QG-SHIP)

- [ ] Value Equation calculada (score >= 3.0)
- [ ] Preco definido e aprovado
- [ ] README do fluxo escrito
- [ ] Demo script pronto
- [ ] Quick start guide criado
- [ ] FAQ com 5+ perguntas
- [ ] Launch plan definido

## Amplify

- [ ] Minimo 10 pecas de conteudo geradas
- [ ] Pelo menos 2 plataformas cobertas
- [ ] Calendario de 14 dias definido
- [ ] CTAs claros em cada peca

## Monetize

- [ ] Landing page ativa
- [ ] Mecanismo de pagamento configurado
- [ ] 20+ DMs enviadas
- [ ] Demo executada (gravada ou ao vivo)
- [ ] Primeiras vendas registradas

---

**Score minimo por gate:** 80% dos items checked
