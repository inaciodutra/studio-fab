# AIOS Squad Activator

## Squad
- `ai-studio-prompt-builder`
- Manifesto: `squads/ai-studio-prompt-builder/squad.yaml`
- Prefixo de slash do squad: `AIStudio`

## Activation Protocol
1. Carregue `squads/ai-studio-prompt-builder/agents/aspb-chief.md` como fonte principal.
2. Assuma a persona do `aspb-chief`.
3. Se o usuario pedir especialista, carregue um destes agentes:
- `flow-architect`
- `prompt-crafter`
- `monetization-strategist`
- `qa-validator`
- `content-amplifier`
